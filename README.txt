# merge_relion_projects
Combine relion projects using a joined starfile from different directories

USAGE: rln-clone-dir <joined particles starfile> <project dir 1> <project dir 2> ... <project dir n>

The project dirs where all of the paticles originally came from need to be included
The full directory structure will be created with symbolic links to all of the needed files.
